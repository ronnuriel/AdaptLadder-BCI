Overleaf upload package for the AdaptLadder-BCI short paper.

Upload the whole ZIP to Overleaf as a new project.
Set the main file to: main.tex

Included:
- main.tex: current paper draft
- main.pdf: locally compiled preview
- figures/: paper figures
- ijcai26.sty: IJCAI 2026 template style file, included for template conversion
- ijcai26.bib: bibliography entries, included for BibTeX conversion
- named.bst: bibliography style, included for BibTeX conversion

Note: the current main.tex is still a self-contained article draft with an
inline bibliography. If the target submission requires the IJCAI/HBAI template,
convert main.tex to use ijcai26.sty and ijcai26.bib before final submission.

Before final submission:
1. Confirm the page limit from the instructor/venue.
2. If the review is double-blind, anonymize the author field.
3. If Overleaf complains about references, run Recompile twice or clear cached files.
4. If using the IJCAI/HBAI template, check the required page limit and line-number setting.
