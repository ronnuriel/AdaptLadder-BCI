AdaptLadder HBAI/IJCAI Full-Paper Overleaf Package
==================================================

Upload this ZIP to Overleaf as a new project.

Main file:
  main.tex

Included:
  main.tex          - expanded IJCAI/HBAI-formatted full-paper draft
  ijcai26.sty       - IJCAI 2026 style file
  ijcai26.bib       - bibliography entries
  named.bst         - bibliography style
  figures/          - all figures referenced by main.tex
  main.pdf          - local PDF preview compiled with pdflatex/bibtex

Notes:
  - The paper is currently anonymous in the IJCAI template.
  - This is the longer/full-paper version; confirm the target page limit.
  - If the venue asks for camera-ready/no line numbers, comment out \linenumbers in main.tex.
  - If Overleaf asks for the main document, choose main.tex.
